# 力扣 102. 二叉树的层序遍历

## 题目内容

给你二叉树的根节点 `root`，按 **层序遍历** 返回节点值。也就是：

-   第 0 层从左到右
    
-   第 1 层从左到右
    
-   第 2 层从左到右
    
-   ...
    
-   最终返回 `List<List<Integer>>`
    

例如：

```text
输入：root = [3,9,20,null,null,15,7]
输出：[[3],[9,20],[15,7]]
```

---

## 核心思路

层序遍历最直接的方式就是：

-   **按层一层层处理**
    
-   每处理一层，就把这一层的节点值收集到一个 `List<Integer>` 中
    
-   再把下一层节点加入待处理结构中
    

这题常见有两类写法：

1.  **DFS（递归）按深度收集**：次优，适合理解“层号”的概念
    
2.  **BFS（队列）层序遍历**：最优，也是面试最推荐写法
    

下面按 **次优到最优** 排序说明。

---

# 解法一：DFS 递归按层收集

## 思路

虽然题目叫层序遍历，大家第一反应是 BFS，但其实也可以用 DFS 做。

DFS 遍历时，额外记录当前节点所在的层数 `depth`：

-   如果当前来到的是一个新层，就先创建一个新的 `List<Integer>`
    
-   再把当前节点值加入 `result.get(depth)` 中
    
-   继续递归左右子树，层数传 `depth + 1`
    

### 为什么可行？

因为每个节点都知道自己在第几层，只要按层号把值塞进去，最后得到的结构就是层序结果。

---

## 适用场景

-   你对递归更熟
    
-   想训练“树 + 层号”的组合写法
    
-   后续题目如果要“按层统计最大值/平均值/最左值”，这种写法也能迁移
    

---

## 代码实现（Java）

```java
import java.util.ArrayList;
import java.util.List;

class Solution {
    public List<List<Integer>> levelOrder(TreeNode root) {
        List<List<Integer>> result = new ArrayList<>();
        dfs(root, 0, result);
        return result;
    }

    private void dfs(TreeNode node, int depth, List<List<Integer>> result) {
        // 递归终止条件：空节点直接返回
        if (node == null) {
            return;
        }

        // 如果当前层还不存在，先创建这一层的列表
        if (depth == result.size()) {
            result.add(new ArrayList<>());
        }

        // 将当前节点值加入对应层
        result.get(depth).add(node.val);

        // 先递归左子树，再递归右子树，保证同层从左到右
        dfs(node.left, depth + 1, result);
        dfs(node.right, depth + 1, result);
    }
}
```

> LeetCode 已经提供 `TreeNode`，通常不需要自己写。其定义一般如下：

```java
public class TreeNode {
    int val;
    TreeNode left;
    TreeNode right;
    TreeNode() {}
    TreeNode(int val) { this.val = val; }
    TreeNode(int val, TreeNode left, TreeNode right) {
        this.val = val;
        this.left = left;
        this.right = right;
    }
}
```

---

## 复杂度分析

-   **时间复杂度：**`O(n)`
    
    -   每个节点访问一次
        
-   **空间复杂度：**`O(n)`
    
    -   结果集本身要存所有节点
        
    -   递归栈最坏为 `O(n)`，平衡树时约 `O(log n)`
        

---

## 细节技巧

### 1）为什么 `depth == result.size()` 时创建新层？

因为 `result` 中现有层数刚好等于当前访问到的最大层数。

比如第一次到第 2 层时：

-   `result.size()` 可能正好是 2
    
-   说明第 2 层还没建
    
-   需要 `add(new ArrayList<>())`
    

---

### 2）为什么先左后右？

题目要求同层节点从左到右输出。

DFS 虽然不是按层访问，但在同一层内，先递归左子树再递归右子树，可以保证同层加入结果时顺序正确。

---

### 3）DFS 的缺点

这题本质是“层序”，DFS 虽然能做，但并不是最直观方案。面试里如果题目是层序遍历，**优先写 BFS** 更符合题意，也更容易让面试官快速认可。

---

# 解法二：BFS 队列层序遍历（最优）

## 思路

这是这题最标准的写法。

使用一个队列：

1.  先把根节点入队
    
2.  每次取出当前队列中的所有节点，这些节点就是同一层
    
3.  把它们的值收集起来
    
4.  再把它们的左右孩子加入队列
    
5.  重复直到队列为空
    

### 关键点

每轮开始先记录：

```java
int size = queue.size();
```

这个 `size` 表示“当前层节点个数”。

然后只循环 `size` 次，就能保证这一轮只处理一层，不会把下一层混进去。

---

## 适用场景

-   面试中的标准答案
    
-   真正意义上的层序遍历
    
-   后续很多题都直接复用这个模板：
    
    -   107 二叉树的层序遍历 II
        
    -   199 二叉树的右视图
        
    -   637 二叉树的层平均值
        
    -   515 在每个树行中找最大值
        

---

## 代码实现（Java）

```java
import java.util.ArrayList;
import java.util.LinkedList;
import java.util.List;
import java.util.Queue;

class Solution {
    public List<List<Integer>> levelOrder(TreeNode root) {
        List<List<Integer>> result = new ArrayList<>();

        // 空树直接返回空结果
        if (root == null) {
            return result;
        }

        Queue<TreeNode> queue = new LinkedList<>();
        queue.offer(root); // 根节点先入队

        while (!queue.isEmpty()) {
            int size = queue.size(); // 当前层的节点数量
            List<Integer> level = new ArrayList<>();

            // 只处理当前层的 size 个节点
            for (int i = 0; i < size; i++) {
                TreeNode node = queue.poll();
                level.add(node.val); // 收集当前层节点值

                // 左孩子存在则入队，留给下一层处理
                if (node.left != null) {
                    queue.offer(node.left);
                }

                // 右孩子存在则入队，留给下一层处理
                if (node.right != null) {
                    queue.offer(node.right);
                }
            }

            // 当前层处理完后加入结果集
            result.add(level);
        }

        return result;
    }
}
```

---

## 复杂度分析

-   **时间复杂度：**`O(n)`
    
    -   每个节点入队出队各一次
        
-   **空间复杂度：**`O(n)`
    
    -   最坏情况下队列中会存一整层节点
        

---

## 细节技巧

### 1）为什么一定要先保存 `size = queue.size()`？

因为当前层处理过程中，会不断把下一层节点加入队列。

如果你直接写成：

```java
for (int i = 0; i < queue.size(); i++)
```

那么 `queue.size()` 会动态变化，可能导致当前层和下一层混在一起。

---

### 2）为什么 `Queue` 常用 `LinkedList`？

在 Java 里常见写法：

```java
Queue<TreeNode> queue = new LinkedList<>();
```

因为它支持：

-   `offer()` 入队
    
-   `poll()` 出队
    
-   `peek()` 查看队首
    

这是面试里最常见写法。

---

### 3）为什么说这是最优写法？

因为题目本身就是“按层处理”，BFS 天然匹配题意：

-   思维最直观
    
-   代码最稳定
    
-   最容易扩展成“层级统计类”题目
    

---

# 两种解法对比

| 解法  | 时间复杂度 | 空间复杂度 | 是否推荐面试首写 | 特点  |
| --- | --- | --- | --- | --- |
| DFS 按深度收集 | O(n) | O(n) | 一般  | 递归优雅，但不够直观 |
| BFS 队列层序遍历 | O(n) | O(n) | 强烈推荐 | 题意匹配，模板价值高 |

---

# 面试时推荐回答方式

如果面试官问这题，建议你这样组织：

1.  **先说题意**
    
    -   需要按层输出二叉树节点
        
2.  **直接给 BFS**
    
    -   用队列保存待处理节点
        
    -   每轮固定处理当前层节点数
        
3.  **强调关键点**
    
    -   `size = queue.size()` 用来锁定当前层
        
4.  **补充边界**
    
    -   `root == null` 返回空列表
        
5.  **最后给复杂度**
    
    -   时间 `O(n)`，空间 `O(n)`
        

这样会显得你思路非常清晰。

# 实际工作/面试场景建议

## 1）面试优先写 BFS 模板

这题属于二叉树基础题中的“模板题”。<br>在面试里，**队列层序模板一定要熟练到能直接默写**，因为它会反复变形：

-   每层求和
    
-   每层最大值
    
-   每层最后一个节点
    
-   锯齿层序遍历
    
-   自底向上层序遍历
    

如果这题你写得很顺，后面很多树题都能秒转。

---

## 2）DFS 作为补充能力展示

如果你已经写完 BFS，还可以补一句：

> 这题也可以用 DFS，记录层号，把节点加入对应层列表。

这会体现你对题目有更完整的理解，而不是只会背模板。

---

## 3）注意不要把“层”与“节点个数变化”写乱

很多候选人的错误点都在这里：

-   没有提前保存 `size`
    
-   导致一层没处理完就把下一层也处理了
    

这是面试官很爱看的细节，因为它能体现你对队列变化过程是否真正理解。

---

## 4）如果时间紧，先写最稳的版本

这题没必要炫技巧。<br>真实面试里最重要的是：

-   正确
    
-   清晰
    
-   易解释
    

所以优先写 **BFS + size 控制层数**，这是最稳妥方案。

# 最终推荐提交版本

```java
import java.util.ArrayList;
import java.util.LinkedList;
import java.util.List;
import java.util.Queue;

class Solution {
    public List<List<Integer>> levelOrder(TreeNode root) {
        List<List<Integer>> result = new ArrayList<>();

        // 边界情况：空树
        if (root == null) {
            return result;
        }

        Queue<TreeNode> queue = new LinkedList<>();
        queue.offer(root); // 根节点入队

        while (!queue.isEmpty()) {
            int size = queue.size(); // 当前层节点数
            List<Integer> level = new ArrayList<>();

            // 按层处理
            for (int i = 0; i < size; i++) {
                TreeNode node = queue.poll();
                level.add(node.val); // 记录当前节点值

                // 下一层节点入队
                if (node.left != null) {
                    queue.offer(node.left);
                }
                if (node.right != null) {
                    queue.offer(node.right);
                }
            }

            // 保存当前层结果
            result.add(level);
        }

        return result;
    }
}
```
