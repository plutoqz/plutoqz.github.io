# 力扣 236 题：二叉树的最近公共祖先

## 解法一（次优）：父指针映射 + 祖先集合

**适用场景**：需要从任意节点快速回溯到根，或后续还会多次查询 LCA（可复用 parent 映射）。<br>**思路**：

1.  用栈遍历树，记录每个节点的父节点 `parent`。
    
2.  从 `p` 向上回溯到根，加入集合 `ancestors`。
    
3.  从 `q` 向上回溯，第一个出现在 `ancestors` 中的节点即为 LCA。
    

**复杂度**：时间 O(n)，空间 O(n)。

```java
import java.util.Map;
import java.util.HashMap;
import java.util.Set;
import java.util.HashSet;
import java.util.Deque;
import java.util.ArrayDeque;

// Definition for a binary tree node.
// class TreeNode {
//     int val;
//     TreeNode left;
//     TreeNode right;
//     TreeNode(int x) { val = x; }
// }

class Solution {
    public TreeNode lowestCommonAncestor(TreeNode root, TreeNode p, TreeNode q) {
        Map<TreeNode, TreeNode> parent = new HashMap<>();
        Deque<TreeNode> stack = new ArrayDeque<>();
        parent.put(root, null); // 根节点父亲为 null
        stack.push(root);

        // 迭代遍历直到找到 p 和 q 的父指针
        while (!parent.containsKey(p) || !parent.containsKey(q)) {
            TreeNode node = stack.pop();
            if (node.left != null) {
                parent.put(node.left, node); // 记录父指针
                stack.push(node.left);
            }
            if (node.right != null) {
                parent.put(node.right, node); // 记录父指针
                stack.push(node.right);
            }
        }

        // 记录 p 的所有祖先
        Set<TreeNode> ancestors = new HashSet<>();
        TreeNode cur = p;
        while (cur != null) {
            ancestors.add(cur);
            cur = parent.get(cur); // 向上回溯
        }

        // q 向上回溯，首次命中即为 LCA
        cur = q;
        while (!ancestors.contains(cur)) {
            cur = parent.get(cur);
        }
        return cur;
    }
}
```

## 解法二（最优）：递归后序遍历

**适用场景**：单次查询、注重空间效率，代码简洁。<br>**思路**：<br>后序遍历：

-   若当前节点为空，返回 null。
    
-   若当前节点是 `p` 或 `q`，返回当前节点。
    
-   递归左右子树：
    
    -   若左右都非空，说明 `p`、`q` 分别在两侧，当前节点即 LCA。
        
    -   否则返回非空的一侧。
        

**复杂度**：时间 O(n)，空间 O(h)，h 为树高（递归栈）。

```java
import java.util.*;

// Definition for a binary tree node.
// class TreeNode {
//     int val;
//     TreeNode left;
//     TreeNode right;
//     TreeNode(int x) { val = x; }
// }

class Solution {
    public TreeNode lowestCommonAncestor(TreeNode root, TreeNode p, TreeNode q) {
        if (root == null) return null; // 空树
        if (root == p || root == q) return root; // 命中目标节点

        TreeNode left = lowestCommonAncestor(root.left, p, q);  // 左子树找
        TreeNode right = lowestCommonAncestor(root.right, p, q); // 右子树找

        if (left != null && right != null) return root; // 两边各找到一个
        return left != null ? left : right; // 返回非空一侧
    }
}
```

## 细节技巧

-   必须用节点引用比较（`root == p`），而不是值比较。
    
-   若题目保证 `p`、`q` 都在树中，递归写法可以直接返回，不必额外校验。
    
-   对于深度极大的树，递归可能导致栈溢出，可改用显式栈（解法一或迭代后序）。

-   面试中优先给出递归后序法，逻辑清晰、空间最优，适合白板。

-   实际工程中若树会频繁查询，需考虑预处理（如 Euler Tour + RMQ）而非单次 DFS。
