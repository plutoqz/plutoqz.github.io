# 力扣 101 题：对称二叉树

## 题目描述

给定一个二叉树的根节点 `root`，检查它是否是轴对称的（即围绕其中心镜像对称）[1][3]。

**示例 1：**

```text
    1
   / \
  2   2
 / \ / \
3  4 4  3
```

输出：true

**示例 2：**

```text
    1
   / \
  2   2
   \   \
   3    3
```

输出：false

---

## 解法一：迭代法（队列）- 次优

### 算法思路

使用队列成对存储需要比较的节点，每次取出两个节点进行比较。将左子树的左孩子与右子树的右孩子、左子树的右孩子与右子树的左孩子分别入队 [3][5]。

### 代码实现

```java
import java.util.LinkedList;
import java.util.Queue;

/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode() {}
 *     TreeNode(int val) { this.val = val; }
 *     TreeNode(int val, TreeNode left, TreeNode right) {
 *         this.val = val;
 *         this.left = left;
 *         this.right = right;
 *     }
 * }
 */
class Solution {
    public boolean isSymmetric(TreeNode root) {
        if (root == null) return true;  // 空树是对称的
        
        Queue<TreeNode> queue = new LinkedList<>();
        queue.offer(root.left);   // 左子树根节点入队
        queue.offer(root.right);  // 右子树根节点入队
        
        while (!queue.isEmpty()) {
            TreeNode leftNode = queue.poll();   // 取出左节点
            TreeNode rightNode = queue.poll();  // 取出右节点
            
            // 两个节点都为空，继续比较下一对
            if (leftNode == null && rightNode == null) {
                continue;
            }
            
            // 一个为空一个不为空，或值不相等，返回false
            if (leftNode == null || rightNode == null || leftNode.val != rightNode.val) {
                return false;
            }
            
            // 外侧节点入队：左节点的左孩子 vs 右节点的右孩子
            queue.offer(leftNode.left);
            queue.offer(rightNode.right);
            
            // 内侧节点入队：左节点的右孩子 vs 右节点的左孩子
            queue.offer(leftNode.right);
            queue.offer(rightNode.left);
        }
        
        return true;
    }
}
```

### 复杂度分析

-   时间复杂度：O(n)，需要遍历所有节点
    
-   空间复杂度：O(n)，队列最多存储 n 个节点
    

### 适用场景

-   树深度较大，担心递归栈溢出
    
-   需要显式控制遍历过程

## 解法二：迭代法（栈）- 次优

### 算法思路

与队列法类似，使用栈代替队列。由于栈是 LIFO，需要注意节点入栈和出栈的顺序 [3][7]。

### 代码实现

```java
import java.util.Stack;

class Solution {
    public boolean isSymmetric(TreeNode root) {
        if (root == null) return true;
        
        Stack<TreeNode> stack = new Stack<>();
        stack.push(root.left);   // 左子树根节点入栈
        stack.push(root.right);  // 右子树根节点入栈
        
        while (!stack.isEmpty()) {
            TreeNode rightNode = stack.pop();  // 先弹出右节点（后入先出）
            TreeNode leftNode = stack.pop();   // 再弹出左节点
            
            if (leftNode == null && rightNode == null) {
                continue;
            }
            
            if (leftNode == null || rightNode == null || leftNode.val != rightNode.val) {
                return false;
            }
            
            // 注意入栈顺序与队列相反
            stack.push(leftNode.left);
            stack.push(rightNode.right);
            stack.push(leftNode.right);
            stack.push(rightNode.left);
        }
        
        return true;
    }
}
```

### 复杂度分析

-   时间复杂度：O(n)
    
-   空间复杂度：O(n)
    

### 适用场景

-   偏好深度优先遍历
    
-   与递归逻辑更接近

## 解法三：递归法 - 最优 ⭐

### 算法思路

定义辅助函数比较两个子树是否互为镜像。核心逻辑：

1.  两个节点都为空 → 对称
    
2.  一个为空一个不为空 → 不对称
    
3.  值不相等 → 不对称
    
4.  递归比较：左节点的左子树 vs 右节点的右子树（外侧），左节点的右子树 vs 右节点的左子树（内侧）[2][3][4]
    

### 代码实现

```java
/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode() {}
 *     TreeNode(int val) { this.val = val; }
 *     TreeNode(int val, TreeNode left, TreeNode right) {
 *         this.val = val;
 *         this.left = left;
 *         this.right = right;
 *     }
 * }
 */
class Solution {
    public boolean isSymmetric(TreeNode root) {
        if (root == null) return true;  // 空树是对称的
        return isMirror(root.left, root.right);  // 比较左右子树
    }
    
    /**
     * 判断两个子树是否互为镜像
     * @param left 左子树节点
     * @param right 右子树节点
     * @return 是否镜像对称
     */
    private boolean isMirror(TreeNode left, TreeNode right) {
        // 终止条件1：两个节点都为空，对称
        if (left == null && right == null) {
            return true;
        }
        
        // 终止条件2：一个为空一个不为空，不对称
        if (left == null || right == null) {
            return false;
        }
        
        // 终止条件3：值不相等，不对称
        if (left.val != right.val) {
            return false;
        }
        
        // 递归比较：外侧对称 && 内侧对称
        // 外侧：left.left vs right.right
        // 内侧：left.right vs right.left
        return isMirror(left.left, right.right) && isMirror(left.right, right.left);
    }
}
```

### 复杂度分析

-   时间复杂度：O(n)，需要遍历所有节点
    
-   空间复杂度：O(h)，h 为树的高度（递归栈空间）
    

### 适用场景

-   **面试首选**：代码简洁，逻辑清晰
    
-   树深度适中（避免栈溢出）
    
-   展示递归思维能力

## 关键细节与技巧

### 1\. 空节点处理

```java
// 错误写法：直接比较值会空指针异常
if (left.val != right.val) return false;

// 正确写法：先判断空节点
if (left == null && right == null) return true;
if (left == null || right == null) return false;
if (left.val != right.val) return false;
```

### 2\. 镜像比较的核心

```text
        left                    right
         / \                    / \
        L   R                  R   L
    
    left.left  镜像  right.right  (外侧)
    left.right 镜像  right.left   (内侧)
```

### 3\. 递归简化写法

```java
// 合并终止条件，代码更简洁
private boolean isMirror(TreeNode left, TreeNode right) {
    if (left == null && right == null) return true;
    if (left == null || right == null || left.val != right.val) return false;
    return isMirror(left.left, right.right) && isMirror(left.right, right.left);
}
```

## 面试/工作场景建议

### 面试场景

| 建议项 | 说明  |
| --- | --- |
| **首选递归** | 代码简洁，展示递归思维，面试官最容易理解 [2][4] |
| **主动提迭代** | 说完递归后补充"也可以用迭代避免栈溢出"，展示全面性 |
| **画图说明** | 在白板/纸上画出镜像对比图，说明外侧/内侧比较逻辑 |
| **边界测试** | 主动提及空树、单节点、完全不对称等测试用例 |

### 工作场景

| 场景  | 推荐方案 | 理由  |
| --- | --- | --- |
| 生产代码 | 递归法 | 可读性最好，便于维护 |
| 树深度未知 | 迭代法 | 避免 StackOverflowError |
| 性能敏感 | 递归法 | 空间复杂度 O(h) 优于迭代 O(n) |
| 代码审查 | 添加注释 | 镜像逻辑需要清晰注释说明 |

### 常见陷阱

1.  **只比较左右孩子值**：忽略结构对称性 [1]
    
2.  **层序遍历误用**：本题不是普通层序遍历，需要成对比较 [3]
    
3.  **空指针异常**：未先判断 null 就访问 val
    
4.  **递归参数错误**：应该是 `(left.left, right.right)` 而非 `(left.left, right.left)`
