# 力扣 226 题：翻转二叉树

## 题目描述

翻转一棵二叉树，即交换每个节点的左右子树，使整棵树呈镜像对称 [1][2]。

```text
输入：     4              输出：    4
        /   \                    /   \
       2     7                  7     2
      / \   / \                / \   / \
     1   3 6   9              9   6 3   1
```

---

## 解法一：递归法（后序遍历）⭐⭐

### 算法思路

1.  递归翻转左子树
    
2.  递归翻转右子树
    
3.  交换当前节点的左右子树 [1][2]
    

### 代码实现

```java
import java.util.*;

/**
 * Definition for a binary tree node.
 */
class TreeNode {
    int val;
    TreeNode left;
    TreeNode right;
    TreeNode(int x) { val = x; }
}

class Solution {
    public TreeNode invertTree(TreeNode root) {
        // 终止条件：空节点直接返回
        if (root == null) {
            return null;
        }
        
        // 递归翻转左右子树（后序：左右中）
        invertTree(root.left);   // 左
        invertTree(root.right);  // 右
        
        // 交换当前节点的左右子树（中）
        swapChildren(root);
        
        return root;
    }
    
    // 辅助方法：交换左右子节点
    private void swapChildren(TreeNode node) {
        TreeNode temp = node.left;    // 暂存左子树
        node.left = node.right;       // 左子树指向原右子树
        node.right = temp;            // 右子树指向原左子树
    }
}
```

### 复杂度分析

-   时间复杂度：O(n)，每个节点访问一次
    
-   空间复杂度：O(h)，h为树高，递归栈空间 [1]
    

### 适用场景

-   代码简洁，易于理解
    
-   树深度不大时推荐使用
    
-   面试中首选展示递归思维

## 解法二：递归法（前序遍历）⭐⭐⭐

### 算法思路

1.  先交换当前节点的左右子树
    
2.  递归翻转左子树
    
3.  递归翻转右子树 [1][2]
    

### 代码实现

```java
class Solution {
    public TreeNode invertTree(TreeNode root) {
        // 终止条件
        if (root == null) {
            return null;
        }
        
        // 前序：先交换当前节点（中）
        TreeNode temp = root.left;      // 保存原左子树
        root.left = root.right;         // 左指向原右
        root.right = temp;              // 右指向原左
        
        // 递归处理子树（左右）
        invertTree(root.left);
        invertTree(root.right);
        
        return root;
    }
}
```

### 复杂度分析

-   时间复杂度：O(n)
    
-   空间复杂度：O(h) [1]
    

### 适用场景

-   与前序遍历逻辑一致
    
-   代码更紧凑，无需辅助方法
    
-   **面试推荐解法**

## 解法三：迭代法（BFS层序遍历）⭐⭐⭐⭐

### 算法思路

1.  使用队列进行层序遍历
    
2.  每层节点出队时交换其左右子树
    
3.  将非空子节点入队继续处理 [1][2]
    

### 代码实现

```java
import java.util.*;

class Solution {
    public TreeNode invertTree(TreeNode root) {
        // 边界条件
        if (root == null) {
            return null;
        }
        
        // 队列用于层序遍历
        Queue<TreeNode> queue = new LinkedList<>();
        queue.offer(root);  // 根节点入队
        
        while (!queue.isEmpty()) {
            TreeNode current = queue.poll();  // 出队当前节点
            
            // 交换左右子树
            TreeNode temp = current.left;
            current.left = current.right;
            current.right = temp;
            
            // 非空子节点入队
            if (current.left != null) {
                queue.offer(current.left);
            }
            if (current.right != null) {
                queue.offer(current.right);
            }
        }
        
        return root;
    }
}
```

### 复杂度分析

-   时间复杂度：O(n)
    
-   空间复杂度：O(w)，w为树的最大宽度 [1]
    

### 适用场景

-   **避免递归栈溢出**（树很深时）
    
-   需要层序处理其他逻辑时可复用
    
-   生产环境更稳健的选择

## 解法四：迭代法（DFS栈模拟）⭐⭐⭐⭐

### 算法思路

1.  使用栈模拟递归过程
    
2.  节点出栈时交换左右子树
    
3.  非空子节点入栈继续处理 [2]
    

### 代码实现

```java
import java.util.*;

class Solution {
    public TreeNode invertTree(TreeNode root) {
        // 边界条件
        if (root == null) {
            return null;
        }
        
        // 栈用于DFS遍历
        Stack<TreeNode> stack = new Stack<>();
        stack.push(root);  // 根节点入栈
        
        while (!stack.isEmpty()) {
            TreeNode node = stack.pop();  // 出栈当前节点
            
            // 交换左右子树
            TreeNode temp = node.left;
            node.left = node.right;
            node.right = temp;
            
            // 子节点入栈（顺序不影响结果）
            if (node.right != null) {
                stack.push(node.right);
            }
            if (node.left != null) {
                stack.push(node.left);
            }
        }
        
        return root;
    }
}
```

### 复杂度分析

-   时间复杂度：O(n)
    
-   空间复杂度：O(h) [2]
    

### 适用场景

-   需要显式控制遍历过程
    
-   理解递归转迭代的经典案例

## ⚠️ 重要注意事项

### 中序遍历的陷阱

**递归中序遍历会导致某些节点翻转两次** [2]：

```java
// ❌ 错误示范 - 中序遍历
invertTree(root.left);   // 翻转左子树
swap(root.left, root.right);  // 交换
invertTree(root.right);  // 此时right已是原left，会重复翻转！
```

如需使用中序，需特殊处理：

```java
// ✅ 中序正确写法
invertTree(root.left);
swap(root.left, root.right);
invertTree(root.left);  // 注意：仍遍历left（已是原right）
```

## 面试与实战建议

| 场景  | 推荐解法 | 理由  |
| --- | --- | --- |
| 面试白板 | 递归前序 | 代码简洁，展示递归思维 [2] |
| 生产环境 | BFS迭代 | 避免栈溢出，更稳健 [1] |
| 树深度大 | BFS/DFS迭代 | 递归可能导致StackOverflowError |
| 需要扩展 | BFS层序 | 便于添加层相关逻辑 |

### 面试加分点

3.  **提及边界条件**（空树、单节点树）
    
4.  **说明递归转迭代思路**展示深度理解
