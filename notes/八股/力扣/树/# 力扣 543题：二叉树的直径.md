# 力扣543题：二叉树的直径

### 题目描述

给定一棵二叉树的根节点，返回该树的**直径**。二叉树的直径是指树中任意两个节点之间最长路径的**长度**，这条路径可能经过也可能不经过根节点。路径长度由节点之间的**边数**表示 [2][3]。

**示例：**

```text
          1
         / \
        2   3
       / \
      4   5
```

返回 **3**，路径为 [4,2,1,3] 或 [5,2,1,3]，边数为3 [3]。

---

## 解法一：暴力DFS（次优解）

### 思路

对每个节点，分别计算其左子树深度和右子树深度，两者之和即为经过该节点的最长路径。遍历所有节点，取最大值 [1][3]。

### 复杂度

-   时间复杂度：O(n²) - 每个节点都要重新计算子树深度
    
-   空间复杂度：O(n) - 递归栈空间
    

### 代码实现

```java
import java.lang.Math;

/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode() {}
 *     TreeNode(int val) { this.val = val; }
 *     TreeNode(int val, TreeNode left, TreeNode right) {
 *         this.val = val;
 *         this.left = left;
 *         this.right = right;
 *     }
 * }
 */
class Solution {
    int maxDiameter = 0;  // 全局变量记录最大直径
    
    public int diameterOfBinaryTree(TreeNode root) {
        if (root == null) return 0;
        
        // 计算经过当前节点的直径（左深度+右深度）
        int currentDiameter = maxDepth(root.left) + maxDepth(root.right);
        maxDiameter = Math.max(maxDiameter, currentDiameter);  // 更新最大值
        
        // 递归遍历左右子树，检查不经过当前节点的路径
        diameterOfBinaryTree(root.left);
        diameterOfBinaryTree(root.right);
        
        return maxDiameter;
    }
    
    // 计算以node为根的子树的最大深度
    private int maxDepth(TreeNode node) {
        if (node == null) return 0;
        return 1 + Math.max(maxDepth(node.left), maxDepth(node.right));  // 深度=1+左右子树最大深度
    }
}
```

### 适用场景

-   理解题目基本思路的入门解法
    
-   树节点数较少时（n < 1000）

## 解法二：优化DFS（最优解）⭐

### 思路

在一次DFS遍历中，同时完成两件事 [2][4]：

1.  返回当前子树的深度（供父节点使用）
    
2.  更新全局最大直径（左深度+右深度）
    

关键洞察：任意路径都可看作由某个节点为起点，从左儿子和右儿子向下遍历的路径拼接而成 [4]。

### 复杂度

-   时间复杂度：O(n) - 每个节点只访问一次
    
-   空间复杂度：O(h) - h为树的高度，递归栈空间
    

### 代码实现

```java
import java.lang.Math;

/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode() {}
 *     TreeNode(int val) { this.val = val; }
 *     TreeNode(int val, TreeNode left, TreeNode right) {
 *         this.val = val;
 *         this.left = left;
 *         this.right = right;
 *     }
 * }
 */
class Solution {
    private int maxDiameter = 0;  // 全局变量记录最大直径（边数）
    
    public int diameterOfBinaryTree(TreeNode root) {
        dfs(root);  // 执行DFS遍历
        return maxDiameter;  // 返回最终结果
    }
    
    /**
     * DFS递归函数
     * @param node 当前节点
     * @return 以node为根的子树的最大深度（节点数）
     */
    private int dfs(TreeNode node) {
        if (node == null) return 0;  // 空节点深度为0
        
        int leftDepth = dfs(node.left);   // 递归计算左子树深度
        int rightDepth = dfs(node.right); // 递归计算右子树深度
        
        // 更新最大直径：经过当前节点的路径长度 = 左深度 + 右深度（边数）
        maxDiameter = Math.max(maxDiameter, leftDepth + rightDepth);
        
        // 返回当前子树深度：左右子树最大深度 + 1（当前节点）
        return Math.max(leftDepth, rightDepth) + 1;
    }
}
```

### 适用场景

-   面试标准答案
    
-   生产环境代码
    
-   大规模树结构（n可达10⁴）

## 关键细节与技巧

### 1\. 深度 vs 直径的区别

| 概念  | 定义  | 计算方式 |
| --- | --- | --- |
| **深度** | 根节点到最远叶子节点的**节点数** | `max(left, right) + 1` |
| **直径** | 任意两节点间最长路径的**边数** | `leftDepth + rightDepth` |

### 2\. 为什么不需要减1？

解法二中 `leftDepth + rightDepth` 直接就是边数，因为：

-   `leftDepth` 表示左子树的深度（节点数），对应边数为 `leftDepth`
    
-   路径从左子树最远节点→当前节点→右子树最远节点，边数恰好为 `leftDepth + rightDepth` [4]
    

### 3\. 全局变量的必要性

直径可能不经过根节点，必须在遍历过程中持续更新最大值，不能只在根节点计算 [1][3]。

## 面试与工作场景建议

### 面试场景

1.  **先说暴力思路**：展示问题分析能力，说明对每个节点计算左右深度之和
    
2.  **指出问题**：时间复杂度O(n²)，存在重复计算
    
3.  **提出优化**：一次DFS同时计算深度和直径，降为O(n)
    
4.  **强调边界**：空树、单节点树、链状树的测试用例
    

### 工作场景

1.  **代码可读性**：添加清晰的注释，方法命名体现意图（如 `calculateDepth` 而非 `dfs`）
    
2.  **线程安全**：若多线程环境，避免使用实例变量，可改用原子类或返回值封装
    
3.  **测试覆盖**：
    
    -   空树：`root = null` → 返回0
        
    -   单节点：`root = [1]` → 返回0
        
    -   链状树：`[1,2,3,4,5]` → 返回4
        
    -   完全二叉树：验证左右子树平衡情况
        

### 扩展变体

-   **最大路径和**（力扣124题）：类似思路，但需处理负值节点，使用 `Math.max(0, dfs(node))` 剪枝 [1]
    
-   **N叉树直径**：遍历所有子节点，取最大的两个深度之和

## 完整可运行代码（含测试）

```java
import java.lang.Math;

class TreeNode {
    int val;
    TreeNode left;
    TreeNode right;
    TreeNode() {}
    TreeNode(int val) { this.val = val; }
    TreeNode(int val, TreeNode left, TreeNode right) {
        this.val = val;
        this.left = left;
        this.right = right;
    }
}

class Solution {
    private int maxDiameter = 0;
    
    public int diameterOfBinaryTree(TreeNode root) {
        maxDiameter = 0;  // 重置全局变量（多次调用时需要）
        dfs(root);
        return maxDiameter;
    }
    
    private int dfs(TreeNode node) {
        if (node == null) return 0;
        
        int leftDepth = dfs(node.left);
        int rightDepth = dfs(node.right);
        
        maxDiameter = Math.max(maxDiameter, leftDepth + rightDepth);
        
        return Math.max(leftDepth, rightDepth) + 1;
    }
    
    // 测试用例
    public static void main(String[] args) {
        Solution solution = new Solution();
        
        // 示例1: [1,2,3,4,5]
        TreeNode root1 = new TreeNode(1);
        root1.left = new TreeNode(2);
        root1.right = new TreeNode(3);
        root1.left.left = new TreeNode(4);
        root1.left.right = new TreeNode(5);
        System.out.println(solution.diameterOfBinaryTree(root1));  // 输出: 3
        
        // 示例2: 单节点
        TreeNode root2 = new TreeNode(1);
        System.out.println(solution.diameterOfBinaryTree(root2));  // 输出: 0
    }
}
```
