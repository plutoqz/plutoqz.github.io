# 力扣 114 题

**题意概述**<br>将二叉树原地展开为“右指针链表”，顺序为先序遍历（root-left-right），并且所有 left 指针置空。

---

## 解法一（次优，易理解）：先序收集再重连

**适用场景**：代码最直观、方便调试；允许额外 O(n) 空间。

**思路**<br>先做先序遍历把节点按顺序放入列表，再顺序重连 right，left 置空。

**复杂度**

-   时间：O(n)
    
-   空间：O(n)
    

**代码（Java）**

```java
import java.util.*;

class Solution {
    public void flatten(TreeNode root) {
        if (root == null) return;
        List<TreeNode> list = new ArrayList<>();
        preorder(root, list);
        for (int i = 1; i < list.size(); i++) {
            TreeNode prev = list.get(i - 1);
            TreeNode curr = list.get(i);
            prev.left = null;          // 关键：清空左指针
            prev.right = curr;         // 关键：右指针串联
        }
    }

    private void preorder(TreeNode node, List<TreeNode> list) {
        if (node == null) return;
        list.add(node);                // 先序：先记录根
        preorder(node.left, list);
        preorder(node.right, list);
    }
}
```

## 解法二（更优，常用）：迭代栈先序，原地重连

**适用场景**：面试常见方案，空间 O(h)，结构清晰。

**思路**<br>用栈模拟先序遍历，每弹出一个节点就接到前一个节点的 right 上，left 置空。

**复杂度**

-   时间：O(n)
    
-   空间：O(h)（h 为树高）
    

**代码（Java）**

```java
import java.util.*;

class Solution {
    public void flatten(TreeNode root) {
        if (root == null) return;

        Deque<TreeNode> stack = new ArrayDeque<>();
        stack.push(root);
        TreeNode prev = null;

        while (!stack.isEmpty()) {
            TreeNode curr = stack.pop();
            if (prev != null) {
                prev.left = null;     // 关键：断开左指针
                prev.right = curr;    // 关键：连接右指针
            }

            // 先压右再压左，保证先序
            if (curr.right != null) stack.push(curr.right);
            if (curr.left != null) stack.push(curr.left);

            prev = curr;
        }
    }
}
```

## 解法三（最优，O(1) 空间）：Morris 变形

**适用场景**：对空间要求严格，展示指针操作能力。

**思路**<br>对每个节点，如果有左子树，就找到左子树最右节点（前驱），<br>把当前节点的右子树接到前驱的右边，再把左子树搬到右边并断开左。

**复杂度**

-   时间：O(n)
    
-   空间：O(1)
    

**代码（Java）**

```java
import java.util.*;

class Solution {
    public void flatten(TreeNode root) {
        TreeNode curr = root;
        while (curr != null) {
            if (curr.left != null) {
                // 找到左子树最右节点
                TreeNode pred = curr.left;
                while (pred.right != null) {
                    pred = pred.right;
                }
                // 将原右子树接到前驱右侧
                pred.right = curr.right;
                // 左子树整体搬到右侧
                curr.right = curr.left;
                curr.left = null; // 关键：断开左指针
            }
            curr = curr.right; // 按链表方向推进
        }
    }
}
```

---

## 细节技巧与易错点

1.  **必须清空 left 指针**，否则不满足题意。
    
2.  先序顺序要求“根-左-右”，栈解法一定“先压右再压左”。
    
3.  Morris 解法中，**前驱是左子树的最右节点**，不是中序前驱的概念。
    
4.  注意不要丢失原 right 子树，必须先挂到前驱的 right 上。
