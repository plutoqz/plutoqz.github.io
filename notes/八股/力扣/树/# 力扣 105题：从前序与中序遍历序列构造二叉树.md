# 力扣105题：从前序与中序遍历序列构造二叉树

## 题目描述

给定两个整数数组 `preorder` 和 `inorder`，其中 `preorder` 是二叉树的**前序遍历**，`inorder` 是同一棵树的**中序遍历**，请构造二叉树并返回其根节点 [1][3]。

**示例：**

```text
输入: preorder = [3,9,20,15,7], inorder = [9,3,15,20,7]
输出: [3,9,20,null,null,15,7]
```

**核心特性：**

-   前序遍历：`[根节点, [左子树前序], [右子树前序]]`
    
-   中序遍历：`[[左子树中序], 根节点, [右子树中序]]` [3][4]
    

---

## 解法一：基础递归法（次优）

**思路：** 每次递归创建新的数组切片，直观但效率较低 [4]。

**时间复杂度：** O(n²) - 每次查找根节点位置需要O(n)<br>**空间复杂度：** O(n²) - 创建大量新数组

```java
import java.util.*;

/**
 * Definition for a binary tree node.
 */
class TreeNode {
    int val;
    TreeNode left;
    TreeNode right;
    TreeNode(int x) { val = x; }
}

class Solution {
    public TreeNode buildTree(int[] preorder, int[] inorder) {
        // 递归终止条件：数组为空
        if (preorder.length == 0 || inorder.length == 0) {
            return null;
        }
        
        // 前序遍历第一个元素是根节点
        int rootVal = preorder[0];
        TreeNode root = new TreeNode(rootVal);
        
        // 在中序遍历中找到根节点位置
        int rootIndex = 0;
        for (int i = 0; i < inorder.length; i++) {
            if (inorder[i] == rootVal) {
                rootIndex = i;
                break;
            }
        }
        
        // 划分左右子树数组
        // 左子树：前序[1, rootIndex+1)，中序[0, rootIndex)
        int[] leftPre = Arrays.copyOfRange(preorder, 1, rootIndex + 1);
        int[] leftIn = Arrays.copyOfRange(inorder, 0, rootIndex);
        
        // 右子树：前序[rootIndex+1, end)，中序[rootIndex+1, end)
        int[] rightPre = Arrays.copyOfRange(preorder, rootIndex + 1, preorder.length);
        int[] rightIn = Arrays.copyOfRange(inorder, rootIndex + 1, inorder.length);
        
        // 递归构建左右子树
        root.left = buildTree(leftPre, leftIn);
        root.right = buildTree(rightPre, rightIn);
        
        return root;
    }
}
```

**适用场景：** 理解算法原理、小规模数据、教学演示

## 解法二：优化递归法（最优⭐）

**思路：** 使用索引范围避免数组拷贝 + HashMap加速根节点查找 [3][6][7]。

**时间复杂度：** O(n) - 每个节点只处理一次<br>**空间复杂度：** O(n) - HashMap存储 + 递归栈

```java
import java.util.*;

/**
 * Definition for a binary tree node.
 */
class TreeNode {
    int val;
    TreeNode left;
    TreeNode right;
    TreeNode(int x) { val = x; }
}

class Solution {
    // 前序遍历数组
    private int[] preorder;
    // 中序遍历值到索引的映射
    private Map<Integer, Integer> inorderMap;
    // 前序遍历当前处理位置
    private int preIndex = 0;
    
    public TreeNode buildTree(int[] preorder, int[] inorder) {
        this.preorder = preorder;
        this.inorderMap = new HashMap<>();
        
        // 构建中序遍历的值-索引映射，O(1)查找根节点位置
        for (int i = 0; i < inorder.length; i++) {
            inorderMap.put(inorder[i], i);
        }
        
        // 递归构建，初始范围为整个中序数组
        return build(0, inorder.length - 1);
    }
    
    /**
     * 递归构建二叉树
     * @param inLeft 中序遍历左边界（包含）
     * @param inRight 中序遍历右边界（包含）
     * @return 当前子树的根节点
     */
    private TreeNode build(int inLeft, int inRight) {
        // 基线条件：中序范围为空，返回null
        if (inLeft > inRight) {
            return null;
        }
        
        // 取前序遍历当前值作为根节点
        int rootVal = preorder[preIndex++];
        TreeNode root = new TreeNode(rootVal);
        
        // 从HashMap获取根节点在中序遍历中的位置
        int inRootIndex = inorderMap.get(rootVal);
        
        // 关键：先构建左子树，再构建右子树
        // 因为preIndex是全局递增的，必须按前序遍历顺序处理
        root.left = build(inLeft, inRootIndex - 1);   // 左子树中序范围
        root.right = build(inRootIndex + 1, inRight); // 右子树中序范围
        
        return root;
    }
}
```

**适用场景：** 面试标准答案、生产环境、大规模数据

## 解法三：迭代法（进阶）

**思路：** 使用栈模拟递归过程，利用前序和中序的特性 [4]。

**时间复杂度：** O(n)<br>**空间复杂度：** O(h) - 栈空间，h为树高

```java
import java.util.*;

/**
 * Definition for a binary tree node.
 */
class TreeNode {
    int val;
    TreeNode left;
    TreeNode right;
    TreeNode(int x) { val = x; }
}

class Solution {
    public TreeNode buildTree(int[] preorder, int[] inorder) {
        // 边界条件
        if (preorder == null || preorder.length == 0) {
            return null;
        }
        
        // 创建根节点
        TreeNode root = new TreeNode(preorder[0]);
        // 栈存储待处理右子树的节点
        Deque<TreeNode> stack = new ArrayDeque<>();
        stack.push(root);
        
        // 中序遍历指针
        int inorderIndex = 0;
        
        // 从第二个节点开始遍历前序数组
        for (int i = 1; i < preorder.length; i++) {
            TreeNode node = stack.peek();
            int preorderVal = preorder[i];
            
            // 情况1：栈顶节点值 != 中序当前值，说明是左子节点
            if (node.val != inorder[inorderIndex]) {
                node.left = new TreeNode(preorderVal);
                stack.push(node.left);
            } 
            // 情况2：栈顶节点值 == 中序当前值，说明需要找右子节点
            else {
                // 弹出所有已完成的左子树节点
                while (!stack.isEmpty() && stack.peek().val == inorder[inorderIndex]) {
                    node = stack.pop();
                    inorderIndex++;
                }
                // 当前节点是最后一个弹出节点的右子节点
                node.right = new TreeNode(preorderVal);
                stack.push(node.right);
            }
        }
        
        return root;
    }
}
```

**适用场景：** 避免递归栈溢出、理解遍历本质、进阶面试

## 关键细节与技巧

### 1\. 索引计算核心公式

```text
左子树节点数 = inRootIndex - inLeft
前序左子树范围：[preIndex + 1, preIndex + leftSize]
前序右子树范围：[preIndex + leftSize + 1, preEnd]
中序左子树范围：[inLeft, inRootIndex - 1]
中序右子树范围：[inRootIndex + 1, inRight]
```

### 2\. HashMap优化的必要性

-   无HashMap：每次查找根节点O(n)，总O(n²)
    
-   有HashMap：查找根节点O(1)，总O(n) [3][6]
    

### 3\. 递归顺序陷阱

```java
// 正确：先左后右，符合前序遍历顺序
root.left = build(inLeft, inRootIndex - 1);
root.right = build(inRootIndex + 1, inRight);

// 错误：先右后左会打乱preIndex顺序
```
