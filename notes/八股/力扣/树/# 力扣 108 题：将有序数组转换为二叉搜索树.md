# 力扣 108 题：将有序数组转换为二叉搜索树

## 题目描述

给定一个升序排列的整数数组 `nums`，将其转换为一棵**高度平衡**的二叉搜索树 [1][2]。

**高度平衡**：每个节点的左右两个子树的高度差的绝对值不超过 1 [2][5]。

---

## 核心思路

有序数组本质上是二叉搜索树**中序遍历**的结果 [2][5]。为了保持高度平衡，需要让左右子树的节点数量尽可能相等，因此选择**数组中间元素作为根节点**，然后递归处理左右子数组 [1][2][5]。

**关键步骤**：

1.  取数组中间位置元素作为当前根节点
    
2.  左半部分递归构建左子树
    
3.  右半部分递归构建右子树
    
4.  递归终止条件：区间为空时返回 null [1][2][5]
    

---

## 解法一：递归法（左闭右闭区间）⭐推荐

```java
import java.util.*;

/**
 * Definition for a binary tree node.
 */
class TreeNode {
    int val;
    TreeNode left;
    TreeNode right;
    TreeNode() {}
    TreeNode(int val) { this.val = val; }
    TreeNode(int val, TreeNode left, TreeNode right) {
        this.val = val;
        this.left = left;
        this.right = right;
    }
}

class Solution {
    public TreeNode sortedArrayToBST(int[] nums) {
        // 调用递归函数，初始区间为 [0, nums.length-1]
        return buildBST(nums, 0, nums.length - 1);
    }
    
    /**
     * 递归构建平衡二叉搜索树
     * @param nums 有序数组
     * @param left 左边界（包含）
     * @param right 右边界（包含）
     * @return 构建的子树根节点
     */
    private TreeNode buildBST(int[] nums, int left, int right) {
        // 递归终止条件：区间为空
        if (left > right) {
            return null;
        }
        
        // 取中间位置作为根节点（防止溢出的写法）
        int mid = left + (right - left) / 2;  // 偶数长度时取靠左的元素
        
        // 创建当前根节点
        TreeNode root = new TreeNode(nums[mid]);
        
        // 递归构建左子树，区间 [left, mid-1]
        root.left = buildBST(nums, left, mid - 1);
        
        // 递归构建右子树，区间 [mid+1, right]
        root.right = buildBST(nums, mid + 1, right);
        
        return root;
    }
}
```

**时间复杂度**：O(n)，每个元素访问一次 [1]<br>**空间复杂度**：O(log n)，递归栈深度（平衡树高度）[1]

**适用场景**：代码简洁，面试首选，易于理解和实现 [2][5]

## 解法二：递归法（左闭右开区间）

```java
import java.util.*;

class Solution {
    public TreeNode sortedArrayToBST(int[] nums) {
        // 左闭右开区间 [0, nums.length)
        return buildBST(nums, 0, nums.length);
    }
    
    /**
     * 递归构建，使用左闭右开区间 [left, right)
     */
    private TreeNode buildBST(int[] nums, int left, int right) {
        // 区间为空的条件
        if (left >= right) {
            return null;
        }
        
        // 只剩一个元素时，直接创建叶子节点（可选优化）
        if (right - left == 1) {
            return new TreeNode(nums[left]);
        }
        
        // 取中间位置
        int mid = left + (right - left) / 2;
        
        // 创建根节点
        TreeNode root = new TreeNode(nums[mid]);
        
        // 左子树区间 [left, mid)，右子树区间 [mid+1, right)
        root.left = buildBST(nums, left, mid);
        root.right = buildBST(nums, mid + 1, right);
        
        return root;
    }
}
```

**时间复杂度**：O(n)<br>**空间复杂度**：O(log n)

**适用场景**：与二分查找的区间表示一致，适合习惯左闭右开写法的开发者 [4][5]

## 解法三：迭代法（队列模拟）

```java
import java.util.*;

class Solution {
    public TreeNode sortedArrayToBST(int[] nums) {
        if (nums == null || nums.length == 0) {
            return null;
        }
        
        // 初始化根节点（先占位）
        TreeNode root = new TreeNode(0);
        
        // 三个队列：节点队列、左边界队列、右边界队列
        Queue<TreeNode> nodeQueue = new LinkedList<>();
        Queue<Integer> leftQueue = new LinkedList<>();
        Queue<Integer> rightQueue = new LinkedList<>();
        
        // 根节点入队
        nodeQueue.offer(root);
        leftQueue.offer(0);
        rightQueue.offer(nums.length - 1);
        
        while (!nodeQueue.isEmpty()) {
            TreeNode curNode = nodeQueue.poll();
            int left = leftQueue.poll();
            int right = rightQueue.poll();
            
            // 计算中间位置
            int mid = left + (right - left) / 2;
            
            // 赋值当前节点
            curNode.val = nums[mid];
            
            // 处理左子树
            if (left <= mid - 1) {
                curNode.left = new TreeNode(0);  // 先占位
                nodeQueue.offer(curNode.left);
                leftQueue.offer(left);
                rightQueue.offer(mid - 1);
            }
            
            // 处理右子树
            if (right >= mid + 1) {
                curNode.right = new TreeNode(0);  // 先占位
                nodeQueue.offer(curNode.right);
                leftQueue.offer(mid + 1);
                rightQueue.offer(right);
            }
        }
        
        return root;
    }
}
```

**时间复杂度**：O(n)<br>**空间复杂度**：O(n)，队列存储所有节点 [5]

**适用场景**：避免递归栈溢出，适合数组极大或递归深度受限的场景 [5]

## 关键细节与技巧

### 1\. 中间元素的选择

当数组长度为偶数时，中间元素有两个，任选其一均可得到平衡树 [1][5]：

```java
// 取靠左的中间元素（向下取整）
int mid = (left + right) / 2;

// 取靠右的中间元素（向上取整）
int mid = (left + right + 1) / 2;

// 随机选择（每次递归随机）
int mid = (left + right + new Random().nextInt(2)) / 2;
```

### 2\. 防止整数溢出

```java
// 推荐写法（防止 left+right 溢出）
int mid = left + (right - left) / 2;

// 位运算优化（效果相同）
int mid = left + ((right - left) >> 1);
```

### 3\. 区间边界处理

| 区间类型 | 终止条件 | 左子树区间 | 右子树区间 |
| --- | --- | --- | --- |
| 左闭右闭 [left, right] | `left > right` | `[left, mid-1]` | `[mid+1, right]` |
| 左闭右开 [left, right) | `left >= right` | `[left, mid)` | `[mid+1, right)` |

## 面试场景建议

### ✅ 推荐做法

1.  **首选递归法（左闭右闭）**：代码最简洁，面试官最容易理解 [2][5]
    
2.  **主动说明中间元素选择**：展示对题目"答案不唯一"的理解 [1][5]
    
3.  **提及溢出问题**：`left + (right - left) / 2` 体现编码规范意识 [4][5]
    
4.  **分析时间复杂度**：O(n) 因为每个节点创建一次 [1]
    

### ⚠️ 注意事项

1.  **不要创建新数组**：用索引操作原数组，避免 O(n²) 空间开销 [5]
    
2.  **递归终止条件要准确**：左闭右闭用 `left > right`，左闭右开用 `left >= right` [4][5]
    
3.  **mid 使用后要排除**：左右子树区间不能包含 mid，否则无限递归 [1]
