## Java逻辑运算符与位运算符详解

---

## 一、概念辨析：逻辑运算符 vs 位运算符

在Java中，需要严格区分两类运算符，它们的作用对象和行为完全不同：

| 运算符类型 | 运算符 | 作用对象 | 短路特性 | 典型用途 |
| --- | --- | --- | --- | --- |
| **逻辑运算符** | `&&`、`\|\|`、`!` | 布尔值(`boolean`) | 有(`&&`、`\|\|`) | 条件判断、流程控制 |
| **位运算符** | `&`、`\|`、`^`、`~` | 整数类型(`int`、`long`等) | 无   | 二进制位操作、数值计算 |

> ⚠️ **关键区别**：逻辑运算符只能用于布尔表达式，位运算符用于整数的二进制位操作 [2][6][8]。

---

## 二、逻辑运算符（Logical Operators）

### 2.1 基本运算规则

| 运算符 | 名称  | 运算规则 | 真值表 |
| --- | --- | --- | --- |
| `&&` | 逻辑与 | 两操作数均为`true`时结果为`true` | `true&&true=true`，其余为`false` |
| `\|\|` | 逻辑或 | 任一操作数为`true`时结果为`true` | `false\|\|false=false`，其余为`true` |
| `!` | 逻辑非 | 对布尔值取反 | `!true=false`，`!false=true` |
| `^` | 逻辑异或 | 两操作数值不同时为`true` | `true^false=true`，相同为`false` [3] |

### 2.2 短路特性（Short-Circuit）

`&&` 和 `\|\|` 具有**短路求值**特性：

```java
// 短路与：第一个为false时，第二个不执行
int a = 5;
boolean b = (a < 4) && (a++ < 10);  // a++不会执行，a仍为5 [2]

// 短路或：第一个为true时，第二个不执行
boolean c = (a > 4) || (a++ < 10);  // a++不会执行 [3]
```

**应用场景**：

-   空指针保护：`if (str != null && str.length() > 0)` [8]
    
-   避免除零错误：`if (b != 0 && a / b > 0)` [8]
    

### 2.3 运算律（布尔代数）

逻辑运算符满足以下运算律 [4]：

| 运算律 | 表达式 |
| --- | --- |
| **交换律** | `A && B = B && A`；`A \|\| B = B \|\| A` |
| **结合律** | `(A && B) && C = A && (B && C)` |
| **分配律** | `A && (B \|\| C) = (A && B) \|\| (A && C)` |
| **幂等律** | `A && A = A`；`A \|\| A = A` |
| **德摩根律** | `!(A && B) = !A \|\| !B`；`!(A \|\| B) = !A && !B` |

## 三、位运算符（Bitwise Operators）

### 3.1 基本运算规则

位运算符对整数的**二进制位**逐位操作 [1][6][10]：

| 运算符 | 名称  | 运算规则 | 示例  |
| --- | --- | --- | --- |
| `&` | 按位与 | 对应位均为1时取1，否则取0 | `3 & 5 = 0011 & 0101 = 0001 = 1` |
| `\|` | 按位或 | 对应位均为0时取0，否则取1 | `3 \| 5 = 0011 \| 0101 = 0111 = 7` |
| `^` | 按位异或 | 对应位相同时取0，相异时取1 | `3 ^ 5 = 0011 ^ 0101 = 0110 = 6` |
| `~` | 按位取反 | 0变1，1变0（单目运算符） | `~3 = ...111100` |

### 3.2 位运算的运算律

**位运算符满足完整的代数运算律** [7][9]：

| 运算律 | 按位与(&) | 按位或(|) | 按位异或(^) | | --- | --- | --- | --- | --- | | **交换律** | `a & b = b & a` | `a \| b = b \| a` | `a ^ b = b ^ a` ✓ | | **结合律** | `(a & b) & c = a & (b & c)` | `(a \| b) \| c = a \| (b \| c)` | `(a ^ b) ^ c = a ^ (b ^ c)` ✓ | | **幂等律** | `a & a = a` | `a \| a = a` | `a ^ a = 0` ⭐ | | **单位元** | `a & -1 = a` | `a \| 0 = a` | `a ^ 0 = a` ⭐ | | **分配律** | `a & (b \| c) = (a & b) \| (a & c)` | `a \| (b & c) = (a \| b) & (a \| c)` | 对&和|均分配 |

## 四、核心问题：对数组所有数进行位运算的结果

### 4.1 异或运算(^)的特殊性质

**异或运算是对数组进行累积运算时最有价值的运算符**，因为它具有以下关键特性 [7][9]：

```text
核心公式：
┌─────────────────────────────────────┐
│  a ^ a = 0      (相同数异或为0)     │
│  a ^ 0 = a      (任何数与0异或不变) │
│  满足交换律和结合律                 │
└─────────────────────────────────────┘
```

### 4.2 数组累积异或的结果规律

对数组中所有元素进行异或运算时：

```java
int result = 0;
for (int num : array) {
    result ^= num;  // 累积异或
}
```

**结果分析**：

| 场景  | 结果  | 原理  |
| --- | --- | --- |
| 所有数都出现**偶数次** | `0` | 每对相同的数异或后抵消为0 |
| 只有一个数出现**奇数次** | 该数本身 | 偶数次的数抵消，只剩奇数次的数 |
| 多个数出现奇数次 | 这些数的异或值 | 无法分离，得到累积异或结果 |

**示例** [7]：

```java
int[] arr = {1, 3, 4, 9, 15, 1, 3, 15, 9};
// 1出现2次，3出现2次，9出现2次，15出现2次，4出现1次
int result = 0;
for (int num : arr) {
    result ^= num;
}
// result = 4（唯一出现奇数次的数）
```

### 4.3 按位与(&)和按位或(|)的累积结果

| 运算符 | 累积运算 | 结果特征 | 是否消除重复 |
| --- | --- | --- | --- |
| `&` | `a & b & c & ...` | 所有数的**公共位**保留 | ❌ 不会消除，但会收敛 |
| `\|` | `a \| b \| c \| ...` | 所有数的**合并位**保留 | ❌ 不会消除，但会收敛 |
| `^` | `a ^ b ^ c ^ ...` | **奇数次**出现的位保留 | ✅ 偶数次抵消 |

**收敛特性**：

-   `&` 运算：随着累积，结果只会**减小或不变**（位只会从1变0）
    
-   `\|` 运算：随着累积，结果只会**增大或不变**（位只会从0变1）
    
-   `^` 运算：结果**不单调**，但相同数会抵消 [1][6]
    

### 4.4 实际应用技巧

#### 技巧1：找出数组中唯一出现奇数次的数

```java
public int findOddTimesNum(int[] arr) {
    int eor = 0;
    for (int num : arr) {
        eor ^= num;
    }
    return eor;  // 出现奇数次的数 [7]
}
```

#### 技巧2：不使用临时变量交换两数

```java
a = a ^ b;
b = a ^ b;  // b = (a^b)^b = a
a = a ^ b;  // a = (a^b)^a = b
```

> ⚠️ 注意：当 `i == j` 时（交换同一位置），会导致值变为0，需加判断 [7][9]

#### 技巧3：判断奇偶性

```java
boolean isOdd = (n & 1) == 1;  // 检查最低位是否为1 [9]
```

#### 技巧4：判断是否为2的幂

```java
boolean isPowerOfTwo = (n > 0) && ((n & (n - 1)) == 0);  // 9]
```

## 五、运算符优先级

在混合表达式中，运算符优先级至关重要 [1][2][6]：

```text
优先级从高到低：
1. 一元运算符：!、~、++、--
2. 算术运算符：*、/、%、+、-
3. 移位运算符：<<、>>、>>>
4. 关系运算符：<、>、<=、>=
5. 相等运算符：==、!=
6. 按位与：&
7. 按位异或：^
8. 按位或：|
9. 逻辑与：&&
10. 逻辑或：||
11. 条件运算符：?:
12. 赋值运算符：=、+=、&=、^= 等
```

## 六、总结

| 问题  | 答案  |
| --- | --- |
| **逻辑运算符能否用于数字** | ❌ 不能，只能用于布尔值；数字需用位运算符 |
| **是否满足运算律** | ✅ 逻辑运算符和位运算符均满足交换律、结合律 |
| **数组累积运算会消除相同数吗** | 仅`^`(异或)会：`a ^ a = 0`；`&`和`\|`不会消除但会收敛 |
| **对数组所有数异或的固定结果** | 出现偶数次的数抵消为0，出现奇数次的数保留 |
| **最佳实践** | 找唯一奇数次元素用`^`；位掩码用`&`/\|`；条件判断用`&&`/\|\|` |
